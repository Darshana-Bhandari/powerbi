This project focuses on Smart Data Discovery using Power BI, where raw data is transformed into meaningful insights through visualization and analysis. The goal is to explore patterns, identify trends, and support data-driven decision-making.

🎯 Objectives
Perform data exploration and cleaning
Identify key patterns and anomalies in the dataset
Handle class imbalance and data variation
Create interactive dashboards for better understanding
Support insights using visual analytics

🛠️ Tools & Technologies
Power BI – Data visualization and dashboard creation
Microsoft Excel / CSV – Data source
Data Cleaning Techniques – Preprocessing and transformation

📂 Project Structure
├── Dataset/              # Raw and cleaned datasets
├── PowerBI_File/         # .pbix file
├── Screenshots/          # Dashboard previews
├── README.md             # Project documentation

📊 Key Features
Interactive dashboards
Data filtering and drill-down capabilities
Visual comparison between Normal Traffic and Attack Traffic
Statistical insights and trend analysis

🔍 Data Insights
Attack traffic shows high variability and extreme values compared to normal traffic
Clear class imbalance exists in the dataset
Significant statistical differences between categories
These findings help in building machine learning models and further analysis

🚀 How to Use
Download the .pbix file from this repository
Open it using Power BI Desktop
Load the dataset if required
Explore dashboards using filters and visuals

📈 Future Improvements
Apply machine learning models for prediction
Improve data balancing techniques
Add more advanced visualizations
